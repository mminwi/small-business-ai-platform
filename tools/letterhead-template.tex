%   4. Update the logo path below to match your actual username/path

% ADJUSTMENTS:
%   Logo width:     Change 1.56in to taste
%   Font size:      Change \fontsize{9}{11} — first number is pt size, second is line spacing

%   Text color:     Change gray}{0.35} — 0 = black, 1 = white, 0.35 = dark gray

%
